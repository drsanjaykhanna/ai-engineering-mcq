# AI Engineering MCQ — Passmedicine for Machine Learning

**205 multiple-choice questions** covering the full AI engineering stack. Built by a doctor who used the Passmedicine method to pass MRCP, MFPM, and medical finals — and then used the same approach to learn AI engineering.

The MCQ method works. If you've used Passmedicine, PassTest, or any question bank to learn clinical medicine, you already know this. Repeated testing with detailed explanations beats passive reading. This applies the same pedagogy to AI/ML concepts.

**Try it now:** [drsanjaykhanna.github.io/ai-engineering-mcq](https://drsanjaykhanna.github.io/ai-engineering-mcq)

---

## Install on your phone (2 minutes)

This is a Progressive Web App — it installs like a native app and works offline.

**iPhone / iPad:**
1. Open [drsanjaykhanna.github.io/ai-engineering-mcq](https://drsanjaykhanna.github.io/ai-engineering-mcq) in Safari
2. Tap the Share button (bottom centre)
3. Scroll down → tap **"Add to Home Screen"**
4. Done. It now works offline, exactly like an app.

**Android:**
1. Open the link in Chrome
2. Tap the three-dot menu (top right)
3. Tap **"Install app"** or **"Add to Home Screen"**
4. Done.

No app store, no account, no tracking, no cost.

## Features

- **Spaced repetition** — questions you get wrong come back more often
- **Streak tracking** — daily practice streaks
- **Mastery system** — tracks which topics you've mastered vs. need work
- **Weakest areas mode** — focuses on your gaps
- **Flag questions** — mark ones to revisit
- **Dark mode** — easy on the eyes
- **Fully offline** — works without internet once installed
- **No account required** — all progress saved locally on your device

## Technical details

Zero-dependency Progressive Web App. No React, no npm, no build step, no backend. Pure HTML/CSS/JS with a service worker for offline caching. Total bundle: ~3.7MB including all content.
